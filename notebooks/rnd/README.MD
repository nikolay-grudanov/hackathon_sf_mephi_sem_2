# Блокноты Research and Development

## Описание категории

В данной категории собраны блокноты, которые используются для исследований и разработок (Research and Development) в области семантического текстового поиска. Эти блокноты представляют собой инструменты для разработки и тестирования алгоритмов поиска слов и словосочетаний в документах с учетом их смыслового значения.
